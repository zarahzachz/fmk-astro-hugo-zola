This is an experimental blog testing static site generators Astro, Hugo and Zola

This project will serve as the template from which all other projects are built from. It will provide basic UI and behavioral guidance to speed up development for the other experiements. Note that the content used in those projects will come from another directory, /original-content, and this project will use filler content instead.
